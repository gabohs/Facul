Linguagens de programacao de mitos
Linguagem Ano de criacao Uso Logo
C++ (a melhor) 1983 É uma linguagem extremamente eficiente e versátil utilizada em ramos que vão desde à criação de jogos até o mercado financeiro
Python 1991 Amplamente utilizada em data science, machine learning. Apesar de ser lenta, sua sintaxe mais simples à torna mais acessível
C 1972 A linguagem mãe. Rápida e Versátil, muito utilizada na área de microcontroladores e sistemas operacionais
C# 2002 Usada principalmente para desenvolvimento de jogos e aplicativos. Muito similar à Java
Java 1995 Muito utilizada em softwares empresariais, desenvolvimento android e jogos
Rust 2015 Muito utilizada em sistemas embarcados. Possui uma maneira inovadora de gerenciamento de memória, fazendo com que a mesma seja mais segura em alguns casos quando comparada à línguagens como C e C++
